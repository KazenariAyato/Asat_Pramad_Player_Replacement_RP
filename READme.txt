Asat Pramad Player Replacement

A Minecraft: Bedrock Edition resource pack that replaces the default player model with a custom model inspired by the character Asat Pramad.

Features

- Custom player geometry
- Custom textures
- Preserves standard player animations
- Works as a resource-pack-only modification
- Designed for Minecraft: Bedrock Edition

Installation

1. Download the resource pack.
2. Import the ".mcpack" file into Minecraft: Bedrock Edition.
3. Open Settings > Global Resources or enable the pack in your world settings.
4. Activate the resource pack.
5. Enter the world and the player model will be replaced with the custom model.

Compatibility

- Minecraft: Bedrock Edition
- Singleplayer
- Realms (resource pack support required)
- Dedicated servers that allow custom resource packs

Compatibility may vary between Minecraft versions. Future game updates may affect model rendering and animations.

Known Limitations

Because this pack replaces the player model:

- Some armor pieces may not align perfectly.
- Certain emotes may not display as intended.
- Marketplace cosmetics may not be fully compatible.
- First-person visuals may differ from vanilla Minecraft.
